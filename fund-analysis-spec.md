# 基金分析平台 — 架構設計 & Design Spec（初版）
---

## 一、專案概述

打造一個以「風險調整後績效」為核心的共同基金分析平台，提供：
- 單一基金的指標分析
- 多基金投資組合比較
- 視覺化圖表呈現風險與報酬關係

---

## 二、功能架構

```
平台功能
├── 搜尋 & 篩選基金
├── 單一基金分析頁
│   ├── 基本資訊（規模、費率、類型）
│   ├── 淨值（NAV）走勢圖
│   └── 風險指標 Dashboard
└── 投資組合比較頁
    ├── 多檔基金指標對比表
    ├── 報酬走勢疊圖
    └── 指標雷達圖
```

## 四、指標規格

### 4.1 核心指標（必做）

| 指標 | 公式 | 解讀 |
|------|------|------|
| **Sharpe Ratio** | (Rp - Rf) / σp | 每單位總風險的超額報酬，越高越好 |
| **Sortino Ratio** | (Rp - Rf) / σd | 只懲罰負向波動，比 Sharpe 更適合防禦型基金 |
| **MDD（最大回檔）** | (最低點 - 前高點) / 前高點 | 歷史最大跌幅，越接近 0% 越好 |
| **Beta (β)** | Cov(Rp, Rm) / Var(Rm) | 相對大盤的波動敏感度，=1 與大盤同步 |
| **Alpha (α)** | Rp - [Rf + β(Rm - Rf)] | 扣除市場風險後的超額報酬，>0 代表經理人創造額外價值 |

### 4.2 建議加入的指標

| 指標 | 說明 |
|------|------|
| **Calmar Ratio** | 年化報酬 / \|MDD\|，同時看報酬和最大回檔，直覺好懂 |
| **年化標準差** | 波動率，最基礎的風險衡量 |

### 4.3 計算週期

預設提供 1 年 / 3 年 / 5 年切換，主推 **3 年**（兼顧時效與穩定性）。

### 4.4 公式備查

```
Sharpe Ratio  = (Rp - Rf) / σp
Sortino Ratio = (Rp - Rf) / σd        # σd = 下行標準差（只計算低於 Rf 的部分）
MDD           = (谷底淨值 - 前高淨值) / 前高淨值
Beta (β)      = Cov(Rp, Rm) / Var(Rm)
Alpha (α)     = Rp - [Rf + β × (Rm - Rf)]
Calmar Ratio  = 年化報酬率 / |MDD|

符號說明：
  Rp = 基金報酬率
  Rf = 無風險利率（建議預設台灣 10 年期公債殖利率，約 1.5–2%）
  Rm = 大盤（基準指數）報酬率
  σp = 年化標準差
  σd = 下行標準差
```

## 六、Design Spec

### 6.1 風格定調

金融科技感（FinTech）：簡潔、數據優先，不花俏。

### 6.2 色彩

| 用途 | 色碼 |
|------|------|
| 主色（Header、主按鈕） | `#1E3A5F` 深藍 |
| 次色（連結、互動） | `#2E86AB` 亮藍 |
| 正向（正報酬、Alpha > 0） | `#06A77D` 綠 |
| 負向（負報酬、高 MDD） | `#D62839` 紅 |
| 背景 | `#F5F7FA` 淺灰 |
| 主文字 | `#1A1A2E` |

### 6.3 指標卡元件

```
┌──────────────────────┐
│  Sharpe Ratio   ⓘ   │  ← 點擊顯示說明
│                      │
│     1.23             │  ← 數字放大顯示
│                      │
│  ██████░░  良好      │  ← 評級 Bar
│  同類均值 0.89        │
└──────────────────────┘

評級對照（以 Sharpe 為例）：
  > 2   → 優秀（綠色邊框）
  1–2   → 良好（藍色邊框）
  0–1   → 普通（灰色邊框）
  < 0   → 偏差（紅色邊框）
```

### 6.4 圖表規格

| 圖表 | 使用場景 |
|------|----------|
| Line Chart | NAV 走勢、多基金比較疊圖 |
| Radar Chart | 多基金指標比較 |
| Area Chart | MDD 視覺化（填色顯示回檔區間） |
| Bar Chart | 指標排行 |

## 七、MVP 範疇（建議）

**Phase 1（核心）**
- 基金搜尋 + 列表頁

**Phase 2（進階，有餘力再做）**
- 投資組合比較頁（雷達圖 + 疊圖）
